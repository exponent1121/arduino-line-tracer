# 🚗 아두이노 라인트레이서 제어 프로젝트

적외선(IR) 센서를 활용하여 검은 선을 추적하는 아두이노 기반 자율주행 로봇

## 🛠 사용 기술
* **하드웨어:** 아두이노 Uno, IR 적외선 센서, L293D 모터 드라이버, DC 모터
* **소프트웨어:** C++ (Arduino IDE)

## 📌 핵심 기능
* 2개의 IR 센서 피드백을 실시간으로 받아 DC 모터의 방향과 PWM 속도를 제어.
* 경로 이탈 시 궤도를 수정하는 동적 제어 알고리즘 구현.
